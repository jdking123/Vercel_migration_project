# Accessibility Automation

## Purpose
Guarantee WCAG 2.2 AA compliance across the system.

## Tools
- Axe
- Playwright a11y tests
- Storybook a11y addon

## Required Checks
- Contrast
- Keyboard navigation
- Focus states
- ARIA roles
- Screen reader labels

## Enforcement
- A11y tests run in CI
- Violations block merges
